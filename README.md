picture-matching
===
This is a game,we called "连连看" in chinese.

Picture-matching contains a server and a client, so users can play together through the Internet.
How to run
------------
* start server
* modify /client/set.ini 
```IP=your server ip```
* start client
* login 
```
default user:
	id:1
	password:123

    id:2
    password:123
```
* enjoy : )

This is a very primer project written in MFC.
